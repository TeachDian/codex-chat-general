# Code Of Conduct

Token Ledger is intended to be a practical, community-driven project.

## Expected Behavior

- Be respectful and direct.
- Focus feedback on the code, docs, design, and user impact.
- Assume good faith when asking questions or reviewing work.
- Keep discussions useful for future readers.
- Respect privacy. Do not ask people to share private prompts, source code, customer data, API keys, or sensitive logs.

## Unacceptable Behavior

- Harassment, threats, insults, or personal attacks.
- Publishing someone else's private information.
- Repeated off-topic comments that make project work harder.
- Deliberately submitting malicious code, unsafe instructions, or private data.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct. Repeated or severe violations may lead to blocked participation.
